News Block

This draft page shows the authoring structure for the EDS news block. The block renders 4 cards per row, loads up to 4 rows total, and includes the fixed filters AI, Robotics, Technology, and Innovation.

The GNews API key is exposed in client-side code on a public site. For a production setup, prefer a server-side proxy. This example keeps the authoring simple for local testing.

Authoring Notes

Use the first column for config labels and the second column for values. The block ignores extra rows it does not understand.

Required row: API Key. Optional rows: Heading, Language, Country, and Button Label.
